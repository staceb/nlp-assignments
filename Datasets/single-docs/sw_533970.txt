Drush

Drush is a free and open source software application that allows people to install and change drupal websites. The name "drush" means ""dru"pal "sh"ell". People who install drush can do all these things using the command line, or terminal of a computer. This makes managing a website that uses drupal faster and more efficient. According to GitHub, drush has been downloaded more than 788,000 times as of March, 2016.
