Universal Mobile Telecommunications System

The Universal Mobile Telecommunications System (UMTS) is a third generation mobile cellular system for networks. It is based on the GSM standard and is part of the International Telecommunications Union IMT-2000 standard set. UMTS is developed and maintained by the 3GPP (3rd Generation Partnership Project). It uses wideband code division multiple access (W-CDMA) radio access technology so that users can transfer information faster.
